# DDD-Sample-Web-Api-.NET-6

This repository is a sample of .net6 web Api.

Please go to the below URL to view the APIs
http(s)://<localUrl>/swagger

# APIs:

1. Register Hiker
2. Get all hikers
3. Get by ID
4. Update Location
5. Report Hiker
6. Trade items
7. Generate Report
8. Add inventory items
9. Get Inventory items
